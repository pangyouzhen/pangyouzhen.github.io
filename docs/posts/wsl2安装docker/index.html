更改镜像位置 Link to heading

背景 Link to heading

  1. docker镜像默认是放在C盘的，如果C盘比较小的话是有问题的

解决办法 Link to heading

  • 修改gui配置，先尝试该方法，如果不生效，尝试第二种
  • 创建软连接方式
  1. 将C:\Users\pang\AppData\Local\Docker备份到C:\Users\pang\AppData\Local\Docker_bak，如果中间出现程序占用问题，在任务管理器杀掉docker，如果还有问题建议将wsl也关闭
  2. 在cmd中创建快捷方式,（注意是cmd而不是powershell，git bash没尝试过），命令如下 mklink /j "C:\Users\pang\AppData\Local\Docker" "你的路径"
  3. 尝试启动docker desktop，如果不行，进入下一步
  4. 将备份的内容全部copy到你的路径下，比如我这里就是C:\Users\pang\AppData\Local\Docker_bak 到你的路径下
  5. 重启docker desktop即可
